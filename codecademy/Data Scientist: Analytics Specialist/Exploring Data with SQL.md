## Exploring Data with SQL
`database` is a set of data stored in a computer, usually structured in <em>tables</em>. 

    SELECT *    
    FROM browse
    LIMIT 10;
    
The above code will `SELECT *` all columns `FROM browse` table for the first 10 records (`LIMIT 10`)
